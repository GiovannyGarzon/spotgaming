FALLAS / SOPORTE




Este módulo permite la administración integral de las solicitudes de servicio y mantenimiento. A continuación se detallan las funcionalidades disponibles:

  • Creación de Reportes de Falla: Permite registrar y documentar incidentes o fallas en las máquinas. Cada reporte incluye información detallada sobre el tipo de falla, la máquina afectada y el usuario que reporta.
  • Gestión de Servicios Técnicos: Facilita la creación de solicitudes para servicios técnicos. Los usuarios pueden especificar el tipo de servicio requerido, asignar personal técnico y establecer fechas de atención.
  • Generación de Remisiones: Posibilita la creación de remisiones para el envío de piezas de repuesto o la movilización de máquinas a talleres o instalaciones. Este proceso incluye la captura de datos relevantes como la dirección de envío y el destinatario.
  • Registro de Reparaciones: Permite documentar el proceso de reparación de las máquinas, incluyendo detalles sobre los trabajos realizados, piezas utilizadas y costos asociados.
  • Validación de Solicitudes de Repuesto: Ofrece la opción de validar y aprobar solicitudes de repuestos necesarias para el mantenimiento y reparación de las máquinas. Esto asegura que se cuente con los recursos necesarios para llevar a cabo el servicio técnico.
